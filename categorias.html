{% comment %} O 'site.categories' do Jekyll é um objeto que contém todas as suas categorias. Vamos iterar (fazer um loop) por cada uma delas. {% endcomment %} {% for category in site.categories %} {% comment %} 'category[0]' é o nome da categoria (ex: "Jekyll"). 'category[1]' é a lista de posts que estão nessa categoria. {% endcomment %} {% assign category_name = category[0] %} {% assign posts_in_category = category[1] %}

{{ category_name }}

    {% for post in posts_in_category %}
  • {{ post.title }}
    • {% endfor %}
{% endfor %}
